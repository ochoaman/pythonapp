
CRUD PYTHON WEB

1.Instalar XAMPP.
2. Abrir la consola, habilitar Apache y Mysql
3. Abrir Mysql con el icono MYSQL.BAT
4. Crear la base de datos: facturacion. COMANDO:
 create database facturacion
5. Subir el scrip (contiene la tabla EMPLEADO). Correr: SubeBDD.bat
6. Correr APP_python.
7. En el navegador (.6): http://127.0.0.1:5000/
